## Selected projects

### Job Application Assistant Agent
Multi-step AI agent that tailors a resume, cover letter, and LinkedIn outreach for any job in under 30 seconds. Built with **integrity enforcement** — the agent is architecturally prohibited from fabricating skills, and a post-tailoring diff check produces a fully auditable changelog with confidence scores for every change.

[Live demo](https://acepilpil-job-application-agent.hf.space) · [GitHub](https://github.com/aces-14/job-application-agent)

### Market Intelligence Agent
Autonomous research agent built with LangGraph that researches a company from four angles, critiques its own draft against a quality threshold, and **self-revises weak sections in a loop**. Stateful multi-node orchestration that goes beyond simple LLM chaining.

[Live demo](https://acepilpil-market-intelligence-agent.hf.space) · [GitHub](https://github.com/aces-14/market-intelligence-agent)

### Philippine Labor Law RAG Assistant
End-to-end RAG system over the Philippine Labor Code — document ingestion, vector embeddings, semantic retrieval, and grounded answers with source citations from the actual law.

[Live demo](https://acepilpil-labor-law-assistant.hf.space)
